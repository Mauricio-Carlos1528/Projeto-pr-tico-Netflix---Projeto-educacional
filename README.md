# Projeto-pr-tico-Netflix---Projeto-educacional

Estrutura do projeto e instruções rápidas:

- `index.html` — página de seleção de perfis (página inicial).
- `catalogo.html` — catálogo com sliders (usa `js/main.js` e `catalogo.css`).
- `replica.html` — réplica simplificada da interface Netflix (exibe grid e modal de trailers).
- `css/` — contém `STYLE.CSS`, `catalogo.css` e `replica.css`.
- `js/` — scripts do projeto (inclui `main.js`, `data.js`, `utils.js` e `replica.js`).
- `assets/` — imagens e ícones usados localmente.

Para abrir localmente no navegador basta abrir `index.html` ou `catalogo.html` no seu navegador.
